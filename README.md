# Programmers AI Helper

프로그래머스 일반 연습문제의 문제 내용과 Python3 코드를 읽어 ChatGPT 웹에 전달할 한국어 프롬프트를 만드는 개인용 Chrome 확장 프로그램입니다.

OpenAI API, API 키, 백엔드 서버를 사용하지 않습니다. 프롬프트와 코드는 사용자가 직접 누르는 클립보드 버튼을 통해서만 이동하며, 코드 실행과 제출도 자동화하지 않습니다.

![Programmers AI Helper 실행 화면](./assets/실행화면%20예시.png)

## 주요 기능

- 문제 제목, 문제 내용, 현재 Python3 코드 추출
- 실행 결과와 오류 신호 탐지
- 다음 목적의 한국어 프롬프트 생성
  - 문제 쉽게 설명
  - 내 코드 검토와 내 코드 평가 (복수 선택 가능)
  - 디버깅 중간 검토
  - 전체 풀이 요청
  - 시간복잡도 평가
  - 실행 오류 분석
- 프롬프트 수정, 초기화 및 클립보드 복사
- ChatGPT 웹 새 탭 열기
- ChatGPT에서 복사한 Python 코드 블록 추출
- CodeMirror 5 에디터에 코드 삽입 후 값 검증
- 문제 URL별 최근 코드 백업 최대 5개 및 이전 코드 복원
- 확장 아이콘을 누른 마지막 탭 하나에서만 사이드 패널 표시

## 지원 범위

- Chrome 116 이상
- `https://school.programmers.co.kr/learn/courses/{숫자}/lessons/{숫자}` 형태의 일반 연습문제
- Python3
- CodeMirror 5 에디터

실제 채용 시험, 평가, 스킬 체크, 기업 과제 페이지에서는 사용하지 마세요. URL pathname에 `assessment`, `exam`, `test`, `skill-check`, `skill_check`, `skill_checks`, `coding-test`가 포함된 페이지에서는 기능이 비활성화됩니다.

## 설치 및 실행 방법

별도의 빌드나 패키지 설치는 필요하지 않습니다.

1. 이 저장소를 내려받거나 clone합니다.
2. Chrome 주소창에서 `chrome://extensions`를 엽니다.
3. 오른쪽 위의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 누릅니다.
5. `manifest.json`이 있는 `programmers-ai-helper` 폴더를 선택합니다.
6. 프로그래머스 일반 연습문제를 열고 언어를 Python3로 선택합니다.
7. 이미 문제 페이지가 열려 있었다면 페이지를 새로고침합니다.
8. Chrome 툴바에서 **Programmers AI Helper** 아이콘을 누릅니다.

사이드 패널은 확장 아이콘을 누른 마지막 탭에만 활성화됩니다. 다른 탭에서 아이콘을 누르면 이전 탭의 패널은 비활성화됩니다.

코드를 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 새로고침하고, 프로그래머스 탭도 다시 새로고침해야 합니다.

## 사용 방법

### 프롬프트 생성

1. 사이드 패널의 현재 상태에서 페이지, 언어, 문제 제목, CodeMirror 탐지 여부를 확인합니다.
2. 원하는 프롬프트 생성 버튼을 누릅니다.
3. 미리보기 내용을 필요에 맞게 수정합니다.
4. 다시 만들고 싶다면 `초기화`를 누릅니다.
5. `생성된 프롬프트 복사`를 누릅니다.
6. `ChatGPT 열기`를 누른 뒤 ChatGPT 입력창에 직접 붙여넣고 전송합니다.

확장 프로그램은 ChatGPT 입력창을 조작하거나 답변을 읽지 않습니다.

`내 코드 검토`와 `내 코드 평가`는 각각 선택하거나 동시에 선택할 수 있습니다. 원하는 항목을 체크한 뒤 `선택한 코드 피드백 생성`을 누르세요. `내 코드 평가`는 정답 여부와 오류 유형을 먼저 확인하고, 정확성·가독성·효율성·Python다운 표현, 복잡도, 제한사항, 엣지 케이스와 개선 전후 차이를 평가하도록 요청합니다.

`디버깅 중간 검토`는 아직 완성하지 않은 코드와 코드 주석의 패턴 분석, TODO, 가설을 중간 작업물로 전달합니다. 완성 답안처럼 정답 여부를 먼저 판정하거나 전체 풀이로 교체하지 않고, 현재까지 확인한 사실과 추측을 나눈 뒤 다음에 확인할 작은 디버깅 단계를 요청합니다.

### 실행 오류 분석

`실행 오류 분석`은 현재 실행 결과에서 다음 신호가 탐지된 경우에만 활성화됩니다.

- `Traceback`
- `Error`, `Exception`
- 주요 Python 예외
- 테스트 실패
- 런타임 오류 또는 컴파일 오류

프로그래머스에서 코드를 직접 실행한 다음 `현재 정보 새로고침`을 누르세요. 실행 전에는 `실행 결과 없음`, 정상 결과에서는 `오류 신호 없음`으로 표시됩니다.

### ChatGPT 코드 삽입

1. ChatGPT 답변에서 Python 코드 블록의 복사 버튼을 누릅니다.
2. 사이드 패널에서 `클립보드 코드 삽입`을 누릅니다.
3. 확장 프로그램이 적절한 Python 코드 블록을 선택하고 마크다운 표시를 제거합니다.
4. 현재 코드를 먼저 백업한 뒤 CodeMirror에 새 코드를 삽입합니다.
5. 실제 삽입값이 원본과 같은지 검증한 후 결과를 표시합니다.
6. 코드를 확인하고 실행 및 제출은 직접 누릅니다.

여러 코드 블록이 있으면 `def solution`이 포함된 Python 블록을 우선하고, 그다음 가장 긴 Python 블록을 사용합니다. 코드로 판단할 수 없는 클립보드 내용은 삽입하지 않습니다.

### 이전 코드 복원

`이전 코드 복원`을 누르면 현재 문제 URL의 가장 최근 백업을 복원합니다. 복원 직전 코드도 다시 백업되므로 잘못 복원한 경우 한 번 더 되돌릴 수 있습니다.

## 권한

| 권한 | 사용 이유 |
| --- | --- |
| `sidePanel` | 클릭한 탭에 Chrome 사이드 패널을 표시합니다. |
| `storage` | URL별 코드 백업과 활성 패널 탭 정보를 저장합니다. |
| `tabs` | 현재 탭을 확인하고 ChatGPT 탭을 엽니다. |
| `scripting` | 페이지 MAIN world에서 CodeMirror 5 API를 호출합니다. |
| `clipboardRead` | 사용자가 코드 삽입을 눌렀을 때 클립보드를 읽습니다. |
| `clipboardWrite` | 사용자가 프롬프트 또는 디버그 정보 복사를 눌렀을 때 클립보드에 씁니다. |
| `https://school.programmers.co.kr/*` | 지원 페이지의 문제와 에디터에 접근합니다. |

`<all_urls>`, 쿠키, 로그인 세션 관련 권한은 사용하지 않습니다.

## 문제 해결

- **확장 아이콘을 눌러도 패널이 열리지 않음**
  - `chrome://extensions`에서 확장 프로그램을 새로고침하고 문제 탭도 새로고침하세요.
  - 확장 프로그램 오류 목록에 이전 버전 오류가 남아 있다면 삭제한 뒤 다시 시도하세요.
- **지원하지 않는 페이지로 표시됨**
  - 일반 연습문제 URL인지 확인하세요. 채용 시험 및 평가 페이지는 지원하지 않습니다.
- **Python3가 아니라고 표시됨**
  - 프로그래머스 언어 선택 메뉴에서 Python3를 선택한 뒤 `현재 정보 새로고침`을 누르세요.
- **CodeMirror를 찾지 못함**
  - 문제 페이지 로딩이 끝난 뒤 페이지와 확장 프로그램을 새로고침하세요.
- **실행 오류 분석 버튼이 비활성화됨**
  - 프로그래머스에서 코드를 직접 실행하고 오류가 화면에 표시된 뒤 `현재 정보 새로고침`을 누르세요.
- **클립보드를 읽거나 쓰지 못함**
  - 사이드 패널을 한 번 클릭해 포커스를 준 뒤 Chrome의 클립보드 권한을 확인하세요.

프로그래머스 DOM이 변경되어 추출이 실패하면 `디버그 정보 복사` 결과를 확인하고 [selectors.js](./selectors.js)의 후보 selector를 수정하세요.

## 개인정보 보호

- 외부 분석, 광고, 텔레메트리 및 API 요청이 없습니다.
- 문제와 코드는 사용자가 복사 버튼을 누를 때만 클립보드에 기록됩니다.
- 클립보드는 사용자가 코드 삽입 버튼을 누를 때만 읽습니다.
- ChatGPT 페이지의 DOM, 답변, 쿠키 및 로그인 정보를 읽지 않습니다.
- 실행 및 제출 버튼을 자동으로 누르지 않습니다.
